# DeepChan
<img width="1254" height="1254" alt="-5195214002437429764_121" src="https://github.com/user-attachments/assets/eb2546bb-298a-4f5e-ae4f-8d0398cc76d5" />

Минималистичный движок анонимной имиджборды для сетей I2P и Tor.

DeepChan создаётся как лёгкая, самодостаточная и максимально независимая площадка для анонимного общения в privacy-first сетях.

Проект изначально ориентирован на:
- работу в медленных сетях
- минимальное потребление трафика
- отсутствие JavaScript
- серверный рендеринг
- самохостинг
- приватность пользователей

---

# Особенности

## Классическая структура имиджборды

- доски
- треды
- ответы
- каталоги
- цитирование постов
- глобальный каталог
- RSS-ленты

Без лишнего визуального шума и перегруженных интерфейсов.

---

# Полная работоспособность без JavaScript

DeepChan проектируется для пользователей:
- I2P
- Tor Browser
- hardened-браузеров
- текстовых браузеров
- пользователей с отключёнными cookie и JS

Весь основной функционал работает через:
- HTML
- серверный рендеринг
- обычные формы

Без:
- React
- SPA
- внешних CDN
- аналитики
- трекеров

# Обработка файлов

DeepChan автоматически выполняет:

- генерацию превью
- очистку метаданных
- проверку файлов
- вычисление SHA-256

- создание превью видео

# Безопасность

Проект делает упор на минимизацию поверхности атак.

Реализовано:
- Stateless CSRF
- HMAC-трипкоды
- защита от тайминг-атак
- очистка EXIF и metadata
- валидация загрузок
- ограничение частоты запросов
- фильтры слов и regex-фильтры
- отсутствие внешних скриптов
- отсутствие сторонней аналитики

---

# Анонимное радио

DeepChan включает встроенную систему анон-радио.

Пользователи могут:
- загружать аудио в посты
- отправлять треки на модерацию
- формировать общий эфир

Поддерживается:
- Icecast
- очистка metadata
- автоматическая перекодировка
- управление плейлистами

# Философия проекта

## Минимализм

DeepChan намеренно избегает:
- тяжёлых frontend-framework
- сложной микросервисной архитектуры
- облачной зависимости
- перегруженного UI
- ненужных abstraction-слоёв

---

## Privacy-first

Проект создаётся для анонимных сетей, а не адаптируется под них постфактум.

Учитываются:
- высокая задержка
- нестабильные туннели
- низкая скорость передачи данных
- отключённый JavaScript
- отключённые cookie

---

## Самодостаточность

DeepChan старается минимизировать внешние зависимости и оставаться максимально автономным.

# Установка (Ubuntu 22.04 / 24.04)

Клонирование репозитория:

```bash
git clone https://github.com/yogannnn/deepchan.git /opt/deepchan
cd /opt/deepchan
```

Запуск установки:

```bash
chmod +x install.sh
sudo ./install.sh
```

После установки:

```text
http://IP_СЕРВЕРА:5000
```

---

# Данные администратора по умолчанию

```text
Логин: admin
Пароль: admin
```

Обязательно измените их после установки.

Файл настроек:

```text
/opt/deepchan/.env
```

---

# Работа в I2P

Для работы через I2P создайте HTTP Tunnel в i2pd/I2P Router:

```text
Target host: 127.0.0.1
Target port: 5000
```

После этого привяжите destination к DeepChan.

---

# Управление сервисом

```bash
systemctl status deepchan
systemctl restart deepchan
journalctl -u deepchan -f
```
