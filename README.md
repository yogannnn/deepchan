# DeepChan

Анонимная имиджборда для сети i2p, ориентированная на минимализм и безопасность.

## Философия

- Никакого JavaScript. Весь функционал доступен через чистый HTML и серверный рендеринг.
- Минимум внешних зависимостей. Все компоненты самодостаточны.
- Полная анонимность и приватность для пользователей i2p.

## Статус

Проект находится на стадии альфа-тестирования. Возможны баги и нестабильная работа. Используйте с осторожностью.

## Установка

1. Клонируйте репозиторий:
   git clone https://github.com/yogannnn/deepchan.git
   cd deepchan

2. Создайте и активируйте виртуальное окружение:
   python3 -m venv .venv
   source .venv/bin/activate

3. Установите зависимости:
   pip install -r requirements.txt

4. Создайте файл .env с необходимыми переменными (пример):
   SECRET_KEY=your_secret_key
   ADMIN_PASSWORD=your_admin_password
   DEPLOY_MODE=production

5. Запустите сервер:
   python app.py

6. Откройте http://127.0.0.1:5000
