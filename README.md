#  DeepChan

Анонимная имиджборда для сетей i2p и onion, ориентированная на минимализм и безопасность.

---

##  Философия

- Никакого JavaScript. Весь функционал доступен через чистый HTML и серверный рендеринг. Ни одного внешнего скрипта.
- Анонимность по умолчанию. Stateless CSRF, отсутствие кук, защита от тайминг-атак, очистка метаданных файлов.
- Самодостаточность. Минимум зависимостей, всё включено «из коробки».
- i2p/onion first. Движок изначально проектировался для работы в медленных и анонимных сетях.

---

##  Возможности

-  Доски и треды — классическая структура, без лишнего.
-  Загрузка медиа — изображения (WebP, PNG, JPG, GIF), видео (MP4, WebM), аудио (MP3, OGG, FLAC).
-  Анон-радио — загружайте аудио в посты, а лучшие треки после модерации попадают в общий эфир.
-  Защищённые трипкоды — подтверждение личности без регистрации. Для админа — специальная плашка Adm.
-  Капча без JS — собственная реализация, только цифры, обновление по клику или ссылке.
-  Гибкая админка — управление досками, тредами, файлами, банами, фильтрами, настройками.
-  RSS-ленты (скоро) — следите за обновлениями через ридеры.
-  Оптимизация для слабых каналов — gzip, кеширование, ленивая загрузка.

##  Бстрая установка (Ubuntu 22.04/24.04)

Выполните в терминале:

  git clone https://github.com/yogannnn/deepchan.git /opt/deepchan
  cd /opt/deepchan
  chmod +x install.sh
  sudo ./install.sh

После выполнения скрипта:
  - Сайт будет доступен на http://<IP-сервера>:5000
  - Админка: логин admin, пароль admin (не забудьте сменить в /opt/deepchan/.env)

Для работы в i2p добавьте туннель типа http на порт 5000 в конфигурации вашего i2pd/роутера.

Управление сервисом:
  systemctl status deepchan   # состояние
  systemctl restart deepchan  # перезапуск
  journalctl -u deepchan -f   # логи
