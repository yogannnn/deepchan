# DeepChan

![DeepChan](https://github.com/user-attachments/assets/eb2546bb-298a-4f5e-ae4f-8d0398cc76d5)

Минималистичный движок анонимной имиджборды для сетей I2P и Tor.

DeepChan создаётся как лёгкая, самодостаточная и максимально независимая площадка для анонимного общения в privacy-first сетях.

---

## Основные принципы

- работа в медленных сетях
- минимальное потребление трафика
- отсутствие обязательного JavaScript
- серверный рендеринг
- самохостинг
- приватность пользователей

---

## Возможности

### Классическая структура имиджборды

- доски
- треды
- ответы
- каталоги
- глобальный каталог
- RSS-ленты
- цитирование постов

---

### Работа без JavaScript

DeepChan проектируется для пользователей:

- I2P
- Tor Browser
- hardened-браузеров
- текстовых браузеров
- пользователей с отключёнными cookie и JS

Весь основной функционал работает через:

- HTML
- CSS
- серверный рендеринг

---

## Архитектура

Проект постепенно переходит к service-oriented архитектуре.

### Ядро использует:

- `services/boards.py`
- `services/posts.py`
- `services/threads.py`
- `services/media.py`
- `services/search.py`

### Плагины взаимодействуют через hooks/events

Примеры:

```python
app.on("posts.before_create", handler)
app.on("boards.filter_list", handler)
```

## Безопасность

Реализовано:

- Stateless CSRF (HMAC)
- CSP-заголовки
- защита от polyglot-файлов
- удаление EXIF
- валидация изображений
- ограничения размера и разрешения
- rate limit и captcha

Подробнее: см. `SECURITY.md`
